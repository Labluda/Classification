# Classification
# Проект: прогнозирование оттока клиентов телеком-оператора

## Описание проекта
В телекоммуникационной компании существует проблема оттока клиентов.  
Компания намеренна активнее работать с клиентами, которые с высокой вероятностью расторгнут договор.

Цель проекта — построить модель машинного обучения, которая предсказывает, разорвет ли клиент договор в ближайшее время.

Проект выполнен как исследовательская работа и ориентирован на решение задачи прогнозирования оттока клиентов с использованием методов машинного обучения.

---

## Цель и задача
**Цель:**  
Предсказать факт расторжения договора клиентом (отток).

**Задача:**  
Построить модель бинарной классификации, определяющую вероятность ухода клиента на основе его персональных данных и информации об используемых услугах.

**Целевая переменная:**  
`status` — бинарный признак, отражающий факт расторжения договора.

---

## Данные
Данные представлены в виде нескольких CSV-файлов, полученных из разных источников:

- `contract_new.csv` — информация о договоре;
- `personal_new.csv` — персональные данные клиента;
- `internet_new.csv` — данные об интернет-услугах;
- `phone_new.csv` — данные об услугах телефонии.

Все таблицы объединяются по идентификатору клиента `customerID`.

Целевой признак `status` сформирован на основе признака `EndDate`.
